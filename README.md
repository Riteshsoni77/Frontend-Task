# Frontend Task

This project is a full-stack application built with **React.js** for the frontend and **Node.js/Express** for the backend. It includes features such as user authentication, task management (CRUD operations), and a responsive dashboard.

---

## Features

### Frontend
- Built with **React.js**.
- Responsive design using **Material-UI**.
- Protected routes with **JWT-based authentication**.
- Forms with client-side validation.
- Task management dashboard with search and filter functionality.

### Backend
- Built with **Node.js** and **Express**.
- **MongoDB** for database storage.
- JWT-based authentication middleware.
- RESTful APIs for user authentication, profile management, and task CRUD operations.
- Password hashing with **bcrypt**.
